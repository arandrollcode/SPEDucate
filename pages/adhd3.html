Variables & Data Types

Variables are boxes to store data inside

When we want to access the data, we just access the variable.

While naming variables, use lowercase letters!

Write this in an editor message = "hello"

The above code is a variable of some text!


Variables & Data Types

A data type is the “type of thing” stored inside a given variable.

A common example is an integer, or a whole number.

If we want to create a data type with a name of age and a value of 16, write this:

age = 16

You can type your code here! Writing the code in the examples is a fun and easy way to learn!
