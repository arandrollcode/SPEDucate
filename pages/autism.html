Installing Python

How to install Python

Go to the official Python website.

Download the Python installer for your operating system.

Run the installer and follow the on-screen instructions.

Once installed, you can verify the installation by opening a terminal or command prompt and typing python --version.

Python Installation Files
  • Download the Python installation file for your operating system:

  • Follow the directions listed!
  • Now, open IDLE. Use this for all our coding!

Variables & Data Types

Variables & Data Types

A data type is the “type of thing” stored inside a given variable.

A common example is an integer, or a whole number.

If we want to create a data type with a name of age and a value of 16, write this:

age = 16
