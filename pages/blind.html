Welcome to Python

Python is one of the most versatile and popular programming languages.

Its syntax makes it much easier to understand compared to most programming languages.

How to Install Python

Follow these steps to install Python on your system:

  • Go to the official Python website. python.org/downloads
  • Download the Python installer for your operating system.
  • Run the installer and follow the on screen instructions.
  • Once installed, you can verify the installation by opening a terminal or command prompt and typing python --version.

Our First Program

Printing Hello World

Now we're going to create our very first program!

Search for a program named IDLE (Integrated Development and Learning Environment) in your operating system. This program should have downloaded with Python. We will use this as our main text editor.

  • Press Control + N to create a new file.
  • In the file, type the following: print("Hello World!").
  • Save the file by pressing Control + S and enter myFirstProgram as the name of the file.
  • Press F5 to run the file.
  • The words "Hello World!" should now be in the console!
  • Congrats! You just made your first program!

Variables

  • Simply put, variables are boxes to store data inside. When we want to access the data, we just access the variable.
  • There are certain rules for naming variables, but all you need to know for now is that we can use letters and underscores
  • You can create a variable like this: variable_name = some_value

Data Types

  • A data type is the type of thing stored inside a given variable
  • Every variable has a data type, ranging from integer to string to list. We will be looking at only a couple.

Data Type: integer

  • Integers are numbers without any decimals
  • An example of creating a variable that stores an integer: i = 3

Data Type: float

  • Floats are numbers with decimals
  • An example of creating a variable that stores a float: f = 5.7
